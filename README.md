### 💻 Projetos em Destaque

#### 🛡️ [DEMS – Distributed Chain of Custody System](https://github.com/pedronuno-soares/startup)
**Sistema orquestrador de cadeia de custódia distribuída focado em integridade e segurança.**
* **Consenso Distribuído:** Implementação de protocolo BFT-Lite e arquitetura Blockchain (v2) para garantir a imutabilidade dos registos.
* **Alta Disponibilidade:** Orquestração de 3 nós MongoDB isolados em rede e base de dados PostgreSQL via Docker Compose.
* **Segurança Criptográfica:** Verificação de integridade de ficheiros através de hashes SHA-256 e fluxos de auditoria.
* **Stack:** TypeScript, Node.js, Express, MongoDB, PostgreSQL, Docker.

#### 🚆 [RailSpot – Train Station Management](https://github.com/pedronuno-soares/railspot)
**Plataforma Full-Stack para gestão e monitorização em tempo real de infraestruturas ferroviárias.**
* **Geolocalização:** Sistema interativo de reporte baseado em mapas para tracking eficiente de ocorrências.
* **Gestão de Dados:** Implementação de base de dados relacional otimizada para histórico de incidentes.
* **Stack:** React, Node.js, Express, PostgreSQL (PERN Stack).

#### 🎰 [Java Betting Service – Logic & Testing](https://github.com/pedronuno-soares/blip-beta-prep-java)
**Serviço Backend robusto focado em lógica de negócio complexa e fiabilidade.**
* **Qualidade de Software:** Garantia de estabilidade através de testes unitários rigorosos com JUnit 5 e Mockito.
* **Clean Code:** Desenvolvimento orientado a objetos com gestão de dependências via Maven.
* **Stack:** Java, Maven, JUnit 5, Mockito.
